# User Persona – Amazon App

## Persona 1: College Student

- Age: 18–24
- Goal: Buy affordable products quickly
- Behaviour:
  - Uses search more than categories
  - Looks for discounts and offers
  - Checks reviews before buying
- Pain Points:
  - Too many notifications
  - Sometimes confusing product filters
- Expectations:
  - Fast delivery
  - Easy return
  - Clear pricing

## Persona 2: Working Professional

- Age: 25–40
- Goal: Save time while shopping
- Behaviour:
  - Uses repeat orders
  - Relies on Prime delivery
- Pain Points:
  - Irrelevant recommendations
- Expectations:
  - Smooth checkout
  - Reliable delivery tracking

## Learning Outcome
Creating personas helps understand different user needs and design better user experiences.
